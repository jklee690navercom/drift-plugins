Wasserstein Distance

기준 분포와 슬라이딩 윈도우 간 Wasserstein 거리(Earth Mover's Distance)로 drift를 탐지한다.

Wasserstein Distance 알고리즘

개요

Wasserstein 거리(Earth Mover's Distance)는 두 확률 분포 사이의 거리를 측정하는 지표이다. 한 분포를 다른 분포로 변환하는 데 필요한 최소 "작업량"을 의미한다.

원리

  • 기준 구간(reference)의 분포와 슬라이딩 윈도우(test)의 분포를 비교
  • Wasserstein 거리: 두 분포의 CDF 차이의 적분
  • 거리가 임계값을 초과하면 drift로 판단
  • Score = distance / threshold

파라미터

파라미터의미기본값효과
Window Size 슬라이딩 윈도우 크기 50 클수록 안정적이나 지연 증가
Reference Ratio 기준 구간 비율 0.5 전체 데이터에서 기준 분포로 사용할 비율
Threshold Wasserstein 거리 임계값 0.1 작을수록 민감, 클수록 보수적

적합한 상황

  • 분포의 전체적인 형태 변화를 감지하고 싶은 경우
  • 평균뿐만 아니라 분산, 왜도 등의 변화도 포착하고 싶은 경우
  • 분포 간의 직관적인 거리 측정이 필요한 경우

참고 문헌

  • Ramdas, A. et al. (2017). "On Wasserstein Two-Sample Testing and Related Families of Nonparametric Tests." Entropy, 19(2), 47.
