KS Test (Kolmogorov-Smirnov)

두 분포의 CDF를 비교하여 분포 변화를 탐지하는 비모수 검정.

KS Test 알고리즘

개요

Kolmogorov-Smirnov (KS) 검정은 두 표본의 누적분포함수(CDF)를 비교하여 두 표본이 같은 분포에서 왔는지를 판정하는 비모수 검정이다. 정규분포를 가정하지 않으므로 어떤 형태의 분포 변화도 감지할 수 있다.

원리

  • D-statistic: 두 CDF 사이의 최대 수직 거리. D = max|F_ref(x) - F_test(x)|
  • p-value: D-statistic이 우연히 발생할 확률. 작을수록 분포가 다르다는 증거.
  • 판정: p-value < alpha(유의수준)이면 "두 분포는 다르다" (drift 감지)

슬라이딩 윈도우 방식

이 플러그인은 기준 구간(reference)과 슬라이딩 윈도우(test)를 비교한다:

  1. 데이터의 앞부분(reference_ratio 비율)을 기준 구간으로 설정
  2. 나머지 구간에서 window_size 크기의 윈도우를 슬라이딩
  3. 각 윈도우 위치에서 기준 구간과 KS 검정을 수행
  4. p-value가 alpha 미만인 구간을 drift로 판정

예제 데이터

데이터 구성

합성 데이터 350건. 기준 구간에서 분포가 변화하는 시나리오.

구간건수평균표준편차설명
기준 (Reference)200건0.900.02정상 상태
Drift150건0.820.04평균 하락 + 분산 증가

CUSUM 예제와 달리 분산도 증가(0.02→0.04)하므로, 분포의 형태 자체가 바뀐다. KS Test는 이런 복합 변화를 감지할 수 있다.

차트 읽는 법

  • 상단 차트 (Value): 원본 시계열. 빨간 점이 alarm 시점.
  • 중단 차트 (D-statistic): 각 윈도우 위치의 KS D-statistic. 높을수록 분포 차이가 큼.
  • 하단 차트 (p-value): 각 윈도우 위치의 p-value. 빨간 점선(alpha=0.05) 아래이면 drift.
