I-MR Chart (Individual-Moving Range)

개별값-이동범위 제어 차트 기반 이상 탐지. 개별 측정값의 이상치와 변동성 변화를 동시에 감지한다.

I-MR Chart 알고리즘

개요

I-MR 차트는 개별 관측값(Individual)과 연속 관측값 간의 이동범위(Moving Range)를 동시에 모니터링하는 제어 차트이다. 서브그룹을 만들 수 없는 개별 측정값에 적합하다.

I Chart (개별값 차트)

개별 값이 UCL(상한 제어 한계)과 LCL(하한 제어 한계) 사이에 있는지 확인한다.

  • 중심선(CL) = X̄ (전체 평균)
  • UCL = X̄ + 2.66 × MR̄ (MR̄ = 이동범위의 평균)
  • LCL = X̄ - 2.66 × MR̄
  • 2.66 = 3/d2 where d2=1.128 for n=2

MR Chart (이동범위 차트)

연속 관측값의 차이(|x_i - x_{i-1}|)가 정상 범위인지 확인한다.

  • 중심선(CL) = MR̄
  • UCL = 3.267 × MR̄ (D4=3.267 for n=2)
  • LCL = 0

AI 모니터링 적용

confidence score, response latency 등 개별 추론 결과의 이상치 탐지에 활용할 수 있다.

파라미터

파라미터의미기본값효과
reference_ratio 기준 구간 비율. 전체 데이터에서 이 비율만큼을 기준으로 사용. 0.5 기준 구간이 안정적일수록 제어 한계가 정확해진다.

데이터 구성

합성 데이터 300건. AI 모델의 confidence score를 시뮬레이션한다.

구간건수평균표준편차설명

기대 결과

  • I Chart에서 LCL 이하 값 다수 감지
  • MR Chart에서 급격한 변동 감지

차트 읽는 법

  • 상단 차트 (I Chart): 개별 값과 UCL/LCL/CL 라인. 빨간 점이 alarm 시점.
  • 하단 차트 (MR Chart): 이동범위 값과 UCL 라인. 빨간 점이 급격한 변동 시점.
