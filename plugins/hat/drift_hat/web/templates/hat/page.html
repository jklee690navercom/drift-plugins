HAT (Hoeffding Adaptive Tree)

두 윈도우의 평균 차이를 Hoeffding bound로 비교하여 drift를 탐지한다. ADWIN 기반 접근.

HAT 알고리즘

개요

Hoeffding Adaptive Tree(HAT)에서 영감을 받은 ADWIN-like drift 탐지 기법이다. 두 개의 인접한 윈도우(W0, W1)를 유지하면서 평균 차이가 Hoeffding bound를 초과하는지 검사한다.

원리

  • 데이터를 순차적으로 처리하면서 두 윈도우 W0(이전), W1(최근)을 유지
  • Hoeffding bound: ε = R · sqrt(ln(2/δ) / (2·m))
  • 여기서 R은 데이터 범위, m은 두 윈도우 크기의 조화평균
  • |mean(W1) - mean(W0)| > ε 이면 drift 감지
  • Score = |mean_diff| / hoeffding_bound

파라미터

파라미터의미기본값효과
Min Window 최소 윈도우 크기 30 클수록 안정적이나 지연 증가
Delta (δ) Hoeffding bound confidence 0.01 작을수록 보수적 (bound 넓어짐)
Reference Ratio 기준 구간 비율 0.5 테스트 시작점 결정

적합한 상황

  • 이론적 보장이 있는 drift 탐지가 필요한 경우
  • 스트리밍 데이터에서 실시간 변화 감지가 필요한 경우
  • 평균의 갑작스러운 변화를 포착하고 싶은 경우

참고 문헌

  • Bifet, A. and Gavalda, R. (2007). "Learning from Time-Changing Data with Adaptive Windowing." SDM.
  • Hoeffding, W. (1963). "Probability Inequalities for Sums of Bounded Random Variables." JASA, 58(301), 13-30.
