MEWMA (Multivariate EWMA)

EWMA 평활화 후 기준 분포로부터의 이탈도(D² 통계량)로 drift를 탐지한다.

MEWMA 알고리즘

개요

MEWMA(Multivariate Exponentially Weighted Moving Average)는 다변량 관리도 기법을 단변량에 적용한 것이다. EWMA로 시계열을 평활화한 뒤, 평활 값이 기준 분포에서 벗어나는 정도를 D² 통계량으로 측정한다.

원리

  • EWMA 평활화: z(t) = λ·x(t) + (1-λ)·z(t-1)
  • D² 통계량: D²(t) = (z(t) - μ_ref)² / σ²_ref
  • Alarm: D² > chi2(1, 1-α) 이면 drift 감지

파라미터

파라미터의미기본값효과
Lambda (λ) EWMA 평활 계수 0.1 작을수록 과거에 더 큰 가중치, 클수록 최근 값에 민감
Reference Ratio 기준 구간 비율 0.5 전체 데이터에서 기준 분포로 사용할 비율
Alpha (α) 유의수준 0.01 작을수록 보수적 (임계값 높음)

적합한 상황

  • 시계열의 점진적인 평균 이동을 탐지하고 싶은 경우
  • 노이즈가 많아 원본 데이터보다 평활화된 값으로 판단이 필요한 경우
  • CUSUM보다 부드러운 감지를 원하는 경우

참고 문헌

  • Lowry, C.A. et al. (1992). "A Multivariate EWMA Control Chart." Technometrics, 34(1), 46-53.

차트 읽는 법

  • 상단 차트: 원본 값(검정) + EWMA 평활 선(파랑). 빨간 점이 alarm 시점.
  • 하단 차트: D² 통계량(보라). 노란 점선이 chi2 임계값.
  • D²가 임계값을 넘으면 drift가 감지된 것이다.
