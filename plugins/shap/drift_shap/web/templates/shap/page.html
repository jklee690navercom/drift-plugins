SHAP Drift (Feature Importance Change)

Rolling statistics를 feature로 추출하여, 기준 구간과 테스트 윈도우 간 feature 분포 변화를 감지한다.

SHAP Drift 알고리즘

개요

Feature importance 변화 기반 drift 탐지이다. 단변량 시계열에서 rolling statistics(rolling_mean, rolling_std, rolling_diff)를 feature로 추출한 뒤, 각 feature의 분포가 기준 구간과 달라졌는지 KS 검정으로 판단한다.

원리

  • Feature 추출: rolling_mean, rolling_std, rolling_diff
  • 각 feature에 대해 기준 구간 vs 테스트 윈도우 KS 검정 수행
  • 모든 feature에서 유의한 변화가 감지되면 drift alarm
  • Score = max(KS statistic) / 0.5

파라미터

파라미터의미기본값효과
Window Size 슬라이딩 윈도우 크기 50 클수록 안정적이나 지연 증가
Reference Ratio 기준 구간 비율 0.5 전체 데이터에서 기준 분포로 사용할 비율
Alpha (α) 유의수준 0.05 작을수록 보수적

적합한 상황

  • 단순 평균/분산 변화 외에 데이터의 동적 특성 변화를 감지하고 싶은 경우
  • 다양한 통계적 특성의 변화를 종합적으로 판단하고 싶은 경우
  • Feature importance 관점에서 drift를 해석하고 싶은 경우

참고 문헌

  • Lundberg, S.M. and Lee, S.I. (2017). "A Unified Approach to Interpreting Model Predictions." NeurIPS.

차트 읽는 법

  • 상단 차트: 원본 값(검정). 빨간 점이 alarm 시점.
  • 하단 차트: 각 feature의 KS drift score. 높을수록 분포 차이가 크다.
  • 모든 feature score가 높은 구간에서 drift가 감지된다.
