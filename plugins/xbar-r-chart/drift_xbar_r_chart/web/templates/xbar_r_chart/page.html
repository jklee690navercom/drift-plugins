X-bar/R Chart

서브그룹 평균(X-bar)과 범위(R)를 동시에 모니터링하는 제어 차트. 배치 단위 데이터의 평균 이동과 산포 변화를 감지한다.

X-bar/R Chart 알고리즘

개요

X̄-R 차트는 데이터를 서브그룹으로 묶어 각 그룹의 평균(X̄)과 범위(R)를 모니터링하는 제어 차트이다. 배치 단위로 데이터가 들어오거나, 시간 구간별로 그룹을 만들 수 있을 때 적합하다.

X̄ Chart (서브그룹 평균 차트)

서브그룹 평균이 제어 한계 안에 있는지 확인한다.

  • CL = X̅ (전체 서브그룹 평균의 평균)
  • UCL = X̅ + A2 × R̄
  • LCL = X̅ - A2 × R̄
  • A2 = 0.577 for subgroup size n=5

R Chart (서브그룹 범위 차트)

서브그룹 범위(max-min)가 제어 한계 안에 있는지 확인한다.

  • CL = R̄
  • UCL = D4 × R̄ (D4=2.114 for n=5)
  • LCL = D3 × R̄ (D3=0 for n=5)

AI 모니터링 적용

시간대별/배치별 평균 confidence와 산포를 모니터링. 평균은 정상인데 산포가 커지면 모델이 불안정해진 것.

파라미터

파라미터의미기본값효과
subgroup_size 서브그룹 크기. 데이터를 이 크기로 묶어서 분석. 5 크기가 클수록 평균 차트가 안정적이나, 그룹 수가 줄어든다.
reference_ratio 기준 구간 비율. 0.5 기준 구간이 안정적일수록 제어 한계가 정확해진다.

예제 데이터

데이터 구성

합성 데이터 300건을 서브그룹 크기 5로 묶어 60개 그룹 생성.

구간그룹 수설명
정상 구간 (그룹 1~40) 40개 안정적 평균과 범위
이상 구간 (그룹 41~60) 20개 평균 하락 + 범위 증가

기대 결과

  • X̄ Chart에서 평균 이탈 감지
  • R Chart에서 범위 증가 감지

차트 읽는 법

  • 상단 차트 (X̄ Chart): 서브그룹 평균과 UCL/LCL/CL 라인. 빨간 점이 alarm.
  • 하단 차트 (R Chart): 서브그룹 범위와 UCL/CL 라인. 빨간 점이 alarm.
